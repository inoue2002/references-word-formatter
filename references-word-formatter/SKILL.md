# References Word Formatter

PDFや論文タイトルから書誌情報を調べ、Wordの「資料文献の作成」ダイアログにコピペできる形式で出力する。

## ワークフロー

### 1. 論文情報の取得

ユーザーから以下のいずれかを受け取る：
- PDFファイル
- 論文タイトル
- DOI
- URL

PDFの場合はメタデータとテキストから情報を抽出。タイトルのみの場合はWeb検索で論文情報を特定。

### 2. 書誌情報の確認

以下の情報を特定・確認する：
- 著者名（全員）
- タイトル
- 発行元（ジャーナル名/出版社）
- 発行年
- その他利用可能な情報（DOI、ページ数など）

### 3. Word形式で出力

Wordの「資料文献の作成」ダイアログにコピペできる形式で出力する。
フィールドの詳細は [references/word_fields.md](references/word_fields.md) を参照。

**出力例：**

```
【論文/レポート】

執筆者: Yamada, Taro; Suzuki, Hanako
タイトル: 深層学習による自然言語処理の発展
発行元: 人工知能学会論文誌
市区町村: 東京
年: 2023
ページ: 45-67
標準番号: 10.1234/example.2023.001
```

## 注意事項

- 執筆者は「姓, 名」形式、複数人はセミコロン区切り
- 組織が著者の場合は「執筆者(組織/団体)」を使用
- 必須フィールド（*）: 執筆者、タイトル、発行元、市区町村、年
- 情報が不明な場合は空欄にせず、ユーザーに確認を求める
