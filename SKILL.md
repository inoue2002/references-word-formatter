# References Word Formatter

PDFや論文タイトルから書誌情報を調べ、Wordの「資料文献の作成」ダイアログにコピペできる形式で出力する。

## 対応する資料文献の種類

| 種類 | 用途 |
|-----|------|
| 論文/レポート | 学位論文、テクニカルレポート |
| 書籍 | 単行本、教科書 |
| 雑誌/定期刊行物の記事 | ジャーナル論文、学術雑誌 |
| Webサイト | Webページ、オンラインリソース |

## ワークフロー

### 1. 論文情報の取得

ユーザーから以下のいずれかを受け取る：
- PDFファイル
- 論文タイトル
- DOI
- URL

PDFの場合はメタデータとテキストから情報を抽出。タイトルのみの場合はWeb検索で論文情報を特定。

### 2. 資料文献の種類を判定

- ジャーナル論文 → **雑誌/定期刊行物の記事**
- 学位論文・レポート → **論文/レポート**
- 単行本 → **書籍**
- Webページ → **Webサイト**

### 3. Word形式で出力

フィールドの詳細は [references/word_fields.md](references/word_fields.md) を参照。

**出力例（雑誌/定期刊行物の記事）：**

```
【雑誌/定期刊行物の記事】

執筆者: Yamada, Taro; Suzuki, Hanako
タイトル: 深層学習による自然言語処理の発展
雑誌/定期刊行物の名前: 人工知能学会論文誌
年: 2023
巻: 38
号: 4
ページ: 45-67
```

**出力例（書籍）：**

```
【書籍】

著者: Tanaka, Jiro
タイトル: 機械学習入門
発行元: 技術評論社
市区町村: 東京
年: 2022
```

## 注意事項

- 著者/執筆者は「姓, 名」形式、複数人はセミコロン区切り
- 組織が著者の場合は「(組織/団体)」を使用
- 情報が不明な場合は空欄にせず、ユーザーに確認を求める
- 必須フィールドは種類ごとに異なる（word_fields.md参照）
