# References Word Formatter

PDFや論文タイトルから書誌情報を調べ、Microsoft Wordの引用管理機能（資料文献）にコピペできる形式で出力するスキル for [Claude Code](https://docs.anthropic.com/en/docs/claude-code).

## 機能

- PDFファイルから書誌情報を抽出
- 論文タイトル・DOI・URLから情報を検索
- Wordの「資料文献の作成」ダイアログに対応した形式で出力

## 対応する資料文献の種類

| 種類 | 用途 |
|-----|------|
| 論文/レポート | 学位論文、テクニカルレポート |
| 書籍 | 単行本、教科書 |
| 雑誌/定期刊行物の記事 | ジャーナル論文、学術雑誌 |
| Webサイト | Webページ、オンラインリソース |

## インストール

```bash
npx add-skill inoue2002/references-word-formatter
```

## 使い方

Claude Code で以下のような依頼をすると、このスキルが自動的に使用されます：

- 「この論文をWord引用形式に変換して」
- 「参考文献をWordフォーマットに」
- 「PDFから引用情報を抽出して」
- 「このジャーナル論文の引用情報を取得」

## 出力例

**雑誌/定期刊行物の記事：**

```
【雑誌/定期刊行物の記事】

執筆者: Yamada, Taro; Suzuki, Hanako
タイトル: 深層学習による自然言語処理の発展
雑誌/定期刊行物の名前: 人工知能学会論文誌
年: 2023
巻: 38
号: 4
ページ: 45-67
```

**書籍：**

```
【書籍】

著者: Tanaka, Jiro
タイトル: 機械学習入門
発行元: 技術評論社
市区町村: 東京
年: 2022
```
